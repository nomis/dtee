Red Hat Enterprise Linux 10
---------------------------

Save the repository configuration file
:download:`dtee-rhel10.repo <../../yum/dtee-rhel10.repo>`
to ``/etc/yum.repos.d/dtee-rhel10.repo``.

Run the following command::

    yum install dtee
